常见问题

快速解答你的疑惑。

DRX 专注于全栈一体化开发体验，内置模块互操作、安全沙箱和诊断系统。无需额外工具链即可处理 ESM/CJS 混合依赖，并提供毫秒级热重载。
当然。DRX 已在数百家企业的生产环境中稳定运行，正常运行率 99.9%+。Enterprise 计划还提供 SLA 保障和专属技术支持。
DRX 兼容 npm 生态，大多数项目只需安装 DRX CLI 并运行 drx migrate 即可完成迁移。我们提供详细的迁移指南和自动化检测工具。
核心运行时支持 TypeScript 和 JavaScript。通过插件系统可扩展 C#、C++ 等语言的互操作。所有语言均享受统一的模块解析和安全策略。
